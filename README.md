# PCG-Heart-Disease-Detection


## Project Overview

This project uses Convolutional Neural Networks (CNNs) to classify heart sounds from Phonocardiogram (PCG) recordings.

The goal is to automatically detect abnormal heart sounds using deep learning techniques.

## Dataset

PhysioNet Challenge 2016 Heart Sound Dataset

## Technologies

- Python
- TensorFlow / Keras
- NumPy
- Pandas
- Librosa
- Matplotlib

## Methodology

1. Audio preprocessing
2. Mel-Spectrogram generation
3. CNN model training
4. Hyperparameter tuning
5. Model evaluation

## Results

- Accuracy: 90%
- Macro F1-Score: 84%

## Future Improvements

- Data augmentation
- Transfer Learning
- Deployment with Streamlit
